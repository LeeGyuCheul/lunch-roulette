# Lunch Roulette

점심 음식 후보와 벤 담당자를 룰렛으로 정하는 정적 웹앱입니다.

## 기능

- 벤 담당 멤버 룰렛
- 음식 후보 룰렛
- 룰렛에서 후보를 바깥으로 드래그해 제외
- 제외된 후보를 다시 룰렛으로 복귀
- 브라우저 `localStorage` 기반 저장

## 실행

`index.html` 파일을 브라우저에서 열면 됩니다.

## 배포

정적 파일만 사용하므로 GitHub Pages, Firebase Hosting, Netlify, Vercel 등에 바로 배포할 수 있습니다.

```text
index.html
styles.css
app.js
```
